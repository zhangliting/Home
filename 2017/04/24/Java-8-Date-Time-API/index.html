Java 8 Date Time API

Clock为instant,date,time 提供时间，是包含时区的。

java.time.Instant

Instant表示时间线上的一个点。

java.time.LocalDate

LocalDate保存日期部分，不包含时区，是不可变得类。

java.time.LocalTime

LocalTime保存时间部分，不包含时区，是不可变得类。

java.time.LocalDateTime

LocalDateTime同时包括了时间和日期。不包含时区，是不可变得类。
